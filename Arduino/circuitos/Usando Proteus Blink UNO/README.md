# Proteus-Arduino-Blink-UNO
Simples projeto no Proteus, Blink da IDE Arduino, usando Arduino UNO.
Inclui a biblioteca Arduino para o Proteus. Descompacte-a e copie seus arquivos para a subpasta [LIBRARY] do Proteus.
Também uso o Virtual Terminal do Proteus, então, tem suas particularidades interessantes.
