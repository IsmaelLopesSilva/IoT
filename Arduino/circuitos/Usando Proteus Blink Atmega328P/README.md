# Proteus-Arduino-Blink-Atmega328P
Simples projeto no Proteus, Blink da IDE Arduino, usando Atmega328P.
Também o Protheus tem um bug com relação ao Osciloscópio. Quando você 
fecha a janela do Osciloscopio durante a simulação, nas próximas simulações
a janela do Osciloscópio não abre sozinha, então, estou anexando uma janela
de como resetar para que volte normalmente.
